TXT

1. git clone https://github.com/andrman1994/TXT.git - Клонировать репозиторий TXT на локальный компьютер

2. cd TXT - Перейти на локальный репозиторий
   touch new.txt - Внутри локального TXT создать файл “new.txt”

3. git add new.txt - Добавить файл под гит

4. git commit -m "+new.txt" - Закоммитить файл

5. git push - Отправить файл на внешний GitHub репозиторий

6. vim new.txt - Открыть файл new.txt в текстовом редакторе VIM
   i - Начать редактирование

Person Name: Andrei
Person Surname: Zhdaniuk
Person Patronomic: Vitalievich
Person Age: 28
Person number of pets: 0
Person desired salary: 9999999999999999999999999999999999999999999$ изменения в формате TXT

   ESC - Завершить редактирование
   :wq - Завершить работу редактора VIM с сохранением изменений

7. git add . - Добавить файл под гит
   git commit -m "update new.txt" - Закоммитить файл
   git push - Отправить изменения на внешний репозиторий

8. touch preferences.txt - Создать файл preferences.txt

9. vim preferences.txt - Открыть файл preferences.txt в текстовом редакторе VIM
   i - Начать редактирование

Person favourite film: The Dark Knight
Person fovourite series: Game of Thrones
Person favourite dish: Pizza with ham
Person favourite time of the year: Summer
Person desired country to visit: United Arab Emirates - изменения в формате TXT

   ESC - Завершить редактирование
   :wq - Завершить работу редактора VIM с сохранением изменений

10. touch sklls.txt - Создать файл sklls.txt
    vim sklls.txt - Открыть файл sklls.txt в текстовом редакторе VIM
    i - Начать редактирование

Person skill 1: Test-cases
Person skill 2: Terminal
Person skill 3: JSON
Person skill 4: Postman
Person skill 5: SQL - изменения в формате TXT

   ESC - Завершить редактирование
   :wq - Завершить работу редактора VIM с сохранением изменений

11. git add . - Добавить файлы под гит
    git commit -m "+ preferences.txt + sklls.txt" - Закоммитить файлы
    git push - Отправить сразу 2 файла на внешний репозиторий

12. git pull - Синхронизировать внешний и локальный репозиторий JSON (после создания баг репорта на внешнем репозитории)
